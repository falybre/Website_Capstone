Potential Technical Questions
Data Quality & Preprocessing
How did you handle missing values in the MMDA dataset? What challenges did you encounter with data quality?
We identified approximately 3.8% missing values, primarily in location data and timestamps. We employed a combination of techniques: removal for rows with critical missing data, mean/mode imputation for numeric features, and spatial approximation for location data. The key challenge was ensuring data completeness without introducing bias.
Model Selection
Why did you choose XGBoost over other approaches like deep learning models or traditional statistical methods?
We evaluated several models and selected XGBoost due to its superior performance (92% accuracy), ability to handle imbalanced data, interpretability through feature importance, and computational efficiency. While deep learning models like LSTM were considered, XGBoost provided better results with our dataset characteristics and required less computational resources.
Feature Engineering
Could you elaborate on the feature engineering process? Which features contributed most to the model's accuracy?
Our feature engineering included temporal features (hour, day, rush hour indicators), spatial features (location clusters), and categorical encodings. Time of day (35%) and location features (28%) contributed most significantly to model accuracy, as evidenced by our feature importance analysis. The strong correlation (r=0.72) between weekday/weekend patterns also proved highly valuable.
Potential Implementation Questions
Real-World Application
How do you envision integrating this predictive model into existing MMDA traffic management systems?
Integration could occur in phases: first as a decision support system providing predictive analytics to traffic managers, then as an automated component of the existing CCTV monitoring infrastructure. We recommend an API-based service architecture that can send real-time alerts to traffic authorities while maintaining the current workflow with minimal disruption.

Future Work
What future enhancements would you suggest to further improve the model's accuracy and applicability?
We recommend incorporating real-time traffic flow data, weather conditions, and special event information to enhance predictions. Developing a multi-modal prediction system that combines XGBoost with deep learning for time-series analysis could further improve accuracy. Integration with smartphone apps for crowdsourced incident reporting would also enrich the dataset continuously.
